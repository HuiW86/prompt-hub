## Risks & Decisions

- **P1.5 未验证**——键盘 sensor + handle focus 依赖 NSPanel key-window（ADR-014），
  必须实机；有「降级为仅指针」出口。
- **reorder 完整性靠调用方约定**——`reorder_macros` 假设入参是完整可见顺序；
  传部分列表会让未触及行保留旧 order_index → 撞号（靠 created_at tiebreak 兜底）。
  前端 `items.map` 永远传全量，故当前安全（informational #1）。
- **后端不校验非空**——create/update_macro 不挡空 name/content，仅 UI canSave 兜底；
  MCP 够不到 repo-write，唯一调用方是 UI，可接受（informational #2）。
- **dnd-kit 0.x API 漂移**——精确锁 0.4.0，升级前审 changelog。
- **B2 物理分离**——拖动严限区域内；AlignmentPhrase 编辑 UI 不得渗入 Composition/Macro。
- 无 remote——checkpoint 仅本地提交。
