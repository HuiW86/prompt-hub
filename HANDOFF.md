- **M1 IPC**：`commands.rs` 4 个 `#[tauri::command]`（全走 `with_write_conn`）；`lib.rs` invoke_handler 注册
- **M2 前端**：`ipc/types.ts` `orderIndex` + `ipc/index.ts` 4 包装 + `promptStore.ts` 4 action（mutation 后全量 `listScenesWithChildren()` refetch，无乐观更新）；`ScenePanel.tsx` 编辑模式（铅笔开关 + per-subStage 独立 `DragDropProvider` 隔离跨组拖拽 + `PhraseEditor` 含「无分组」下拉 + ConfirmInline）；`ScenePanel.module.css` token 化编辑态样式

## Risks & Decisions
- **migration 0009 forward-only 不可逆**（与 0005-0008 一致）—— backfill SQL 已在 cargo test in-memory 库验证
- **NULL 可空分区** —— backfill 与 reorder 的 NULL `sub_stage_id` 路径仅单测覆盖，未在设备上验证
- **删除留空洞不压缩** —— 读路径只 ORDER BY 不依赖连续值，与 AlignmentPhrase 先例一致，评审确认可接受
- **决策 ②**：update 改 `sub_stage_id` 用单 UPDATE 带子查询追加目标分区末尾，杜绝并发撞 MAX+1
- **命名陷阱**：`Phrase`(scene 绑定) ≠ `AlignmentPhrase`(phase 绑定)；phrases.rs ≠ alignment_phrases.rs
